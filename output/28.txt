                <h1>
                    Как включить JavaScript в вашем веб-браузере</h1>
                <h2 class="intro">
                    В настоящее время почти все веб-страницы используют JavaScript, скриптовый язык программирования, который работает на веб-браузере посетителя. Это делает веб-страницы более функциональными, а если по каким-либо причинам JavaScript отключен, содержание или функциональность веб-страницы может быть ограничена или вовсе не возможна. Здесь вы можете найти инструкции о том, как включить (активировать) JavaScript в пяти наиболее распространенных веб-браузерах.</h2>

                    <h3 id="ie">
                        <span class="icon">&nbsp;</span>Internet Explorer
                    </h3>
                    <ol>
                        <li>
                            В меню веб-браузера нажмите иконку &quot;Tools&quot; и выберите пункт &quot;Internet Options&quot;.</li>
                        <li>
                            В разделе &quot;Internet Options&quot; выберите вкладку &quot;Security&quot;.</li>
                        <li>
                            На вкладке &quot;Security&quot; нажмите кнопку &quot;Custom level...&quot;.</li>
                        <li>
                            После открытия диалогового окна &quot;Security Settings - Internet Zone&quot;, посмотрите раздел &quot;Scripting&quot;.</li>
                        <li>
                            В разделе &quot;Active Scripting&quot; выберите пункт &quot;Enable&quot;.</li>
                        <li>
                            После появления всплывающего окна &quot;Warning!&quot; в котором спрашивается &quot;Are you sure you want to change the settings for this zone?&quot; нажмите кнопку &quot;Yes&quot;.</li>
                        <li>
                            В окне &quot;Internet Options&quot; нажмите кнопку &quot;OK&quot;, что бы закрыть его.</li>
                        <li>
                            Нажмите на кнопку &quot;Refresh&quot; в веб-браузере, чтобы обновить страницу.</li>
                    </ol>

                    <h3 id="chrome">
                        <span class="icon">&nbsp;</span>Google Chrome
                    </h3>
                    <ol>
                        <li>
                            В меню браузера нажмите кнопку &quot;Customize and control Google Chrome&quot; и выберите пункт &quot;Settings&quot;.</li>
                        <li>
                            На странице &quot;Settings&quot; нажмите ссылку  &quot;Show advanced settings...&quot;</li>
                        <li>
                            Под пунктом &quot;Privacy&quot; нажмите на кнопку  &quot;Content settings...&quot;.</li>
                        <li>
                            Когда диалоговое окно откроется, посмотрите на секцию &quot;JavaScript&quot; и поставьте галочку &quot;Allow all sites to run JavaScript (recommended)&quot;.</li>
                        <li>
                            Нажмите кнопку &quot;OK&quot;, чтобы закрыть окно.</li>
                        <li>
                            Закройте вкладку &quot;Settings&quot;.</li>
                        <li>
                            Нажмите на кнопку &quot;Reload this page&quot; для обновления страницы.</li>
                    </ol>
